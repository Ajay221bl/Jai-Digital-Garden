# normalizing a dataset
normalizing a data set means rescaling its numeric features to a common range, typically between $0 \text{ and 1}$


[[Notes/importance of normalizing a dataset\|importance of normalizing a dataset]]


While [[Notes/normalizing a data set\|normalization]] is often used as a general term for scaling, there are two main techniques.

1. [[Notes/min-max scaling\|min-max scaling]]
2. [[Notes/z-score\|z-score standardization]]
