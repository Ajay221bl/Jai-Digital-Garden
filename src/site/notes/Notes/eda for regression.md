# [[Notes/exploratory data analysis\|EDA]] for [[Notes/logistic regression\|logistic regression]]

When preparing to fit a [[Notes/regression model\|regression model]], linear or logistic, [[Notes/exploratory data analysis\|exploratory data analysis]] helps us understand basic properties of each variable, such as counts for [[Notes/categorical variable\|categorical variables]] or [[Notes/probability distributions\|distributions]] for numeric variables; even drive variable selection
