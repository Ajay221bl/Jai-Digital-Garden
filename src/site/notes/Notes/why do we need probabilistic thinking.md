# why do we need probabilistic thinking?

- today there exists _complex systems_ and as the complexities in systems arises the [[Notes/uncertainty\|uncertainty]] factor rises. So to deal with this uncertainty,[[Notes/probabilistic thinking\|probabilistic thinking]] is required.
- it helps in making more _informed decisions_
- we live in _information world_ where data is getting generated every second. The purpose of data is to tell what we don't know already. In other words, it is there to help us tackle with the unknown/ the [[Notes/uncertainty\|uncertain]]. hence to model the [[Notes/uncertainty\|uncertainty]], we require [[Notes/probabilistic thinking\|probabilistic thinking]]
