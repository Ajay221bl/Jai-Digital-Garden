# Multiplication rule in conditional probability



according to [[Notes/conditional probability\|conditional probability]]:
$$P(A|B) = \frac{P(A\cap B)}{P(B)}$$
the multiplication rule is:
$$\huge P(A \cap B) = P(A|B).P(B)$$
