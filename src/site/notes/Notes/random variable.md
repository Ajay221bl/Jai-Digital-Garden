# random variable

a random variable is a [[Notes/function\|function]] as it maps the outcomes from an experiment's [[Notes/sample space\|sample space]]  to a set of [[Notes/real numbers\|real numbers]]. it is denoted by ${X}$
let's take an example: flipping a coin.
- __Experiment__: Flipping a single coin.
- __Sample Space($\ohm$)__: The set of all possible outcomes is {Head, Tails}.
- __Random Variable($X$)__: We can define a random variable $X$ to represent the outcome numerically. For instance:
	- $X(Heads)=1$
	- $X(Tails)=0$

here the random variable $X$ takes the abstract outcomes "Heads" and "Tails" and maps them to the number 1 and 0. This allows us to ask mathematical questions, like "What is the probability that X=1?"
