# regression line
> it is also called the [[Notes/regression line\|line of best fit]]
- it represents the predicted values of the [[Notes/response variable\|dependent variable]].
- it is determined by estimating the [[Notes/regression coefficient\|regression coefficients]]($\beta_{1},\beta_{2},\beta{3}$) and the [[Notes/intercept\|intercept]] ($\beta_{0}$) in a way that minimizes the differences between actual values y and the predicted values of y.

the regression line is a straight line that represents the relationship between the [[Notes/explanatory variables\|independent variable]] (y) and the [[Notes/response variable\|dependent variable]] (x) from a simple linear model.
