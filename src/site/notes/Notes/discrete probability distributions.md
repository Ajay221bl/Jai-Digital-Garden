# discrete probability distributions
discrete probability distributions describe the probabilities of outcomes for a [[Notes/discrete random variables\|discrete random variable]]. The probability of each specific value is between 0 and 1, and the sum of all probabilities is exactly 1.


#### common types of discrete distributions:
1. [[Notes/binomial distribution\|binomial distribution]]
2. [[Notes/poisson distribution\|poisson distribution]]
3. [[bernoulli distribution\|bernoulli distribution]]
