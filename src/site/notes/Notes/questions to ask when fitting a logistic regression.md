# questions to ask when fitting a [[Notes/logistic regression\|logistic regression]]

- Can it predict the [[Notes/probability\|Notes/probability]] of a [[Notes/response variable\|dependent variable]] equaling _yes_ (e.g. a customer will churn, a patient has heart disease)? Logistic regression returns coefficients that can be plugged into an equation to predict binary outcomes.
- Is there a meaningful relationship between the [[Notes/response variable\|dependent variable]] and the one or more [[Notes/explanatory variables\|independent variables]]? [[Notes/logistic regression\|logistic regression]] identifies which [[Notes/explanatory variables\|predictors]] havem and don't have, a [[Notes/statistical significance\|statistically significant]] effect on the [[Notes/probability\|Notes/probability]] of a binary outcome.
- How can it measure the relationship between [[Notes/explanatory variables\|predictors]] and outcomes. [[Notes/logistic regression\|logistic regression]] provides [[quantitative insights\|quantitative insights]] into how changes in the [[Notes/explanatory variables\|predictors]] effect the probability of the binary outcome.
- How does it assess the _strength and direction of of associations_ between variables? The returned coefficient estimates measures the relationships between [[predictors\|predictors]] and the log odds of the binary outcome
