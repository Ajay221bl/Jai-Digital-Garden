# Categorical data
- categorical data represents distinct categories or labels
- for example:
	- colors
	- brand of cars
	- gender
	- types of cars


categorical data helps in classifying or organizing data into groups.

A [[Notes/categorical variable\|categorical variable]] is the named characteristics or the feature that you are measuring, which will then be described by the [[Notes/categorical data\|categorical data]].
