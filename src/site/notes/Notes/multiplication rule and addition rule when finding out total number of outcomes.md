# multiplication rule and addition rule when finding out total number of outcomes



[[Notes/multiplication rule\|multiplication rule]] is used when we want to find the total number of outcomes in case of events that are simultaneous or sequential

[[Notes/addition rule\|addition rule]] is used when we want to find the total number of outcomes in case of events that are [[Notes/mutually exclusive sets or events\|mutually exclusive]]
