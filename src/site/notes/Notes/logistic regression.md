# logistic regression
logistic regression is a [[Notes/supervised learning\|supervised learning]] method for predicting a binary response from one or more [[Notes/explanatory variables\|independent variables]]. It's commonly used for [[classification\|classification]] tasks where the [[Notes/response variable\|dependent variable]] represents two possible categories or classes (e.g. pass or fail, presence or absence).

It estimates the [[Notes/probability\|Notes/probability]] that a given instance belongs to a particular category based in the values of the [[Notes/explanatory variables\|independent variables]] or [[Notes/explanatory variables\|predictors]], using the  [[Notes/logistic function\|logistic function]] (also known as the [[Notes/logistic function\|sigmoid function]]) that maps the output to range between $\text{0 and 1}$.



In logistic regression, the [[Notes/response variable\|dependent variable]], which is typically coded as $\text{0 or 1}$, is modeled as a [[Notes/function\|function]] of the [[Notes/explanatory variables\|independent variables]] using the [[Notes/logistic function\|logistic function]], also known as the [[Notes/logistic function\|sigmoid function]]
