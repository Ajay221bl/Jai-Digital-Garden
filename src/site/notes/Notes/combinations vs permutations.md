# combinations vs permutations

when order doesn't matter, its a [[Notes/combination\|combination]]
when order does matter, its a [[Notes/permutation\|permutation]]



combination example:
when a public address announcer introduces the five starting players before a basketball game, that's a combination, because the sequence of introductions doesn't matter.

permutation example:
the digit lock on a briefcase is a permutation because the specific sequence matters significantly: only 3-7-4 will unlock the briefcase, whereas other arrangement of the same numerals will not work.
