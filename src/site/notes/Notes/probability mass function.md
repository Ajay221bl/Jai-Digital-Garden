# Probability Mass Function
the probability mass function finds the probability that a discrete random variable is exactly equal to some specific value. 

The value of the PMF at a specific point $x$, denoted as $P(X=x)$, is the actual probability of the outcome occurring.


Mathematically the [[Notes/probability mass function\|PMF]] satisfies the following two properties:
1. _Non negativity_:  The PMF assigns non-negative [[Notes/probability\|probabilities]] to each possible outcome, therefore ensuring that all probabilities are greater than or equal to zero.
2. _Summation_: The sum of all probabilities for all possible outcomes of the random variable is equal to 1. In other words $\sum{P(X=x)} = 1$, where the sum is taken over all possible values of x.
