# contingency table
a contingency table, also known as a [[Notes/contingency table\|cross-tabulation]] or [[Notes/contingency table\|crosstab]], is a table that summarizes the relationship between two or more [[Notes/categorical variable\|categorical variables]]. It displays the [[frequency distribution\|frequency distribution]] of the variables, showing how often specific combinations of their values occur

Each cell in the table represents the frequency or count of occurances for a particular combination of classes from the two variables.


more specifically:
- each row in the [[Notes/contingency table\|contingency table]] corresponds to a distinct class of one [[Notes/categorical variable\|categorical variable]]/
- each column corresponds to a distinct class of the other [[Notes/categorical variable\|categorical variable]]
- each cell contains a count or frequency of occurrences that align to a combination of classes from each [[Notes/categorical variable\|categorical variable]].
