# categorical variable
it is the named characteristic or the feature that you are measuring which will then be described by the [[Notes/categorical data\|categorical data]]. It's the _container_ for the [[Notes/categorical data\|categorical data]]. It defines what is being classified.

- __categorical variable__: Hair color
- __categorical data__: The actual entries you collect, such as _Black, Brown, Blonde, Red_.
