# continuous probability distributions

continuous probability distributions describe the [[Notes/probability\|probabilities]] for a [[Notes/continuous random variables\|continuous random variable]], which can take on any value within a given range.

For a [[Notes/continuous probability distributions\|continuous distribution]], the probability of the variable taking on any single specific value is _zero_. Instead, probabilities are calculated for ranges of values using a [[Notes/probability density function\|probability density function]]. The total are under the [[Notes/probability density function\|PDF]] curve is equal to _1_.


#### common types of continuous distributions
1. [[Notes/normal distribution\|normal distribution]] (Gaussian Distribution)
2. [[Notes/uniform distribution\|uniform distribution]]
