# [[Notes/probability density function\|probability density function]] vs [[Notes/probability mass function\|probability mass function]]

The [[Notes/probability density function\|PDF]] for [[Notes/continuous random variables\|continuous random variables]] returns relative likelihoods, but the [[Notes/probability mass function\|PMF]] for [[Notes/discrete random variables\|discrete random variables]] returns actual [[Notes/probability\|probabilities]] of occurrence for each possible value.
